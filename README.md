# Zyvola Finanzas (React + PWA)

Proyecto modular de finanzas con enfoque educativo y visual premium.

## Incluye

- Modulo de calculadoras financieras
- Modulo de guias educativas
- Modulo de glosario visual
- Modulo de economia explicada facil
- Arquitectura reusable en componentes
- Base preparada para PWA (manifest + service worker)

## Uso

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
